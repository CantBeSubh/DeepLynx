# Project Structure

Let me teach you what a project is.

## Nexus Starts with Projects

A project is where your data all is.

- **Press the blue button**: Click upload project button after logging in

## Containers

Every project has a container, that's also where your data is but smaller.

> [!TIP]
>
> The terms `directory` and `folder` may be used interchangeably.
