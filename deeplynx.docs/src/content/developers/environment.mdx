# Setting Up a .env File for PostgreSQL Connection in .NET

Environment variables are a common way to configure applications, especially for sensitive information such as database credentials. This guide will walk you through the steps to set up a `.env` file in your .NET project's root folder to connect to a PostgreSQL database.

## Prerequisites

Before you begin, ensure you have the following:

- A PostgreSQL database up and running.
- Basic knowledge of environment variables.
- A .NET project where you want to store the `.env` file.

## Step 1: Create the .env File

In your project’s root directory, create a file named `.env`. This file will store your PostgreSQL database connection details. Open the `.env` file in a text editor and add the following content:

```plaintext
# PostgreSQL connection details
DB_HOST=localhost
DB_PORT=5432
DB_USER=myuser
DB_PASSWORD=mypassword
DB_NAME=mydatabase
