# Setting Up PostgreSQL

PostgreSQL is a powerful, open-source object-relational database system that uses and extends the SQL language. This guide will walk you through the steps to set up PostgreSQL on your system.

## Prerequisites

Before you begin, ensure you have the following:

- A system running a Unix-based OS (Linux, macOS) or Windows.
- Administrative access to install software.
- Basic command-line knowledge.
