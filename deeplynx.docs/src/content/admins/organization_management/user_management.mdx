# User Management

The Users tab of the Organization Management page has a table containing all active and pending users.
Organization admins have the ability to add, edit, and remove users from organizations.

## Adding Users to Organizations

Follow these steps to add a user to an organization in Nexus:

1. Navigate to the Organization Management page by clicking the slider bar icon on the outtermost toolbar.
2. On the Users tab of the Organization Management page, click the "Invite User" button.
3. Enter the user's email address and hit "Send Invitation".
4. The user will receive an email containing a link to join the organization!

## Editing Users in Organizations

1. Click the pencil icon in the action column of the user management table.
2. Update the user's information in the Edit User modal and hit save.

## Removing Users from Organizations

1. Click the trashcan icon in the action column of the user management table.
2. This will remove the user from the organization.

**Note:** Once a user is removed from the organization, a new invitation is required to regain access.
