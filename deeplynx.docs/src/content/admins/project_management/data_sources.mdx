# Data Sources

## Adding Data Sources

1. Navigate to the Organization Management page by clicking the slider bar icon on the outtermost toolbar.
2. On the Tags and Security Labels tab of the Organization Management page, click the "New Tag" button.

/>

## Editing Data Sources

Organization admins can edit the names of existing organization tags.

1. Click the "Edit" button next to the selected tag inside the Organization Tags widget.

<Image
  src="/docs/images/pre-edit-tag-org.png"
  alt="Pre Editing Tags UI"
  width={800}
  height={600}
/>

## Removing Data Sources

The organization admin has the ability to remove organization tags by clicking the "Delete" button of Organization Tags Widget.

**Note:** Once a data source is archived it can no longer be used. It must be restored to be reapplied to an organization.
