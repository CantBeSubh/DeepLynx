# Data Sources

## Adding Data Sources

1. Navigate to the Project Management page by clicking the slider bar icon on the inner toolbar.
2. On the Data Sources tab of the Project Management page, click the "Add Data Source" button.

<Image
  src="/docs/images/add-data-proj.png"
  alt="Creating Tags UI"
  width={800}
  height={600}
/>

3. Enter the required details for the new data source and click the "Create" button.

<Image
  src="/docs/images/add-data-name-proj.png"
  alt="Creating Tags UI"
  width={800}
  height={600}
/>

4. The new data source will be added to the Data Sources widget of the Project Management page!

## Editing Data Sources

Project admins can edit the name, abbreviation, type, base URI/ Connection, description, and the config of existing project data sources.

1. Click the pencil icon button "Details" next to the selected data source.

<Image
  src="/docs/images/pre-edit-data-proj.png"
  alt="Pre Editing Tags UI"
  width={800}
  height={600}
/>

2. Make changes to the data source fields and click "Save".

<Image
  src="/docs/images/edit-data-proj.png"
  alt="Pre Editing Tags UI"
  width={800}
  height={600}
/>

## Removing Data Sources

1. To remove data sources, click the "Archive" button of the selected data source.

<Image
  src="/docs/images/pre-arch-data-proj.png"
  alt="Removing Tags UI"
  width={800}
  height={600}
/>

**Note:** Once a data source is archived it will have a yellow highlight and can no longer be used.
It must be unarchived to be reapplied to a project.

<Image
  src="/docs/images/arch-proj.png"
  alt="Removing Tags UI"
  width={800}
  height={600}
/>

## Hiding Archived Data Sources
The project admin also has the ability to hide archived projects by switching the toggle "Hide Archived".

<Image
  src="/docs/images/hide-arch-proj.png"
  alt="Removing Tags UI"
  width={800}
  height={600}
/>

Click the unarchive button of the selected data source.
