# Project Settings

Project admins will soon have the ability to customize project banners and unmounted storage locations.
- This can be found on the the Settings tab of the Project Management page.
- If a banner is already set at the organization level, the banner field will be disabled.

**Note:** Customizing project banners and additional unmounted storage locations will be enabled in a future release.
