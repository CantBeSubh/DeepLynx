# User Management

The Users tab of the Project Management page has a table containing all active and pending users.
Project admins have the ability to add, edit, and remove users from projects.

## Adding Users to Projects

1. Navigate to the Project Management page by clicking the slider bar icon on the outtermost toolbar.
2. On the Users tab of the Project Management page, click the "Invite User" button.
3. Select the user's email address and role then click "Add to Project".
4. The user will appear in the users table of the Project Management page!

**Note:** The add user modal populates a drop down of existing users and roles.

## Editing Users in Projects

1. Click the pencil icon in the action column of the user management table.
2. Update the user's information in the Edit User modal and hit save.

<Image
  src="/docs/images/edit-user-org.png"
  alt="Edit User UI"
  width={800}
  height={400}
/>

## Removing Users from Projects

1. Click the trashcan icon in the action column of the user management table.
2. This will remove the user from the project.

**Note:** Once a user is removed from the project, they must be readded to regain access.
