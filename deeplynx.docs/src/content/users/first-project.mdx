# Your First Project

Let's learn how to create and manage your first project in DeepLynx.

## How to Create a Project

There are two ways to create a project in DeepLynx: through the UI and via the API.

When a project is created, three default classes are automatically created and associated with your newly created project:

- **Timeseries**: For time-series data management
- **Report**: For report generation and storage
- **File**: For file management and storage

### Creating a Project in the UI

Follow these steps to create a project through the DeepLynx interface:

1. Click the **Create Project** button on the DeepLynx Home Page
2. Give the project a name (and description if desired)
3. Click the **Create** button and wait to be taken to the home page of your newly created project

### Creating a Project via API

To create a project programmatically using the API:

1. First obtain an API Key ([learn how to get an API key](#))
2. Set up a POST request to `/api/projects/CreateProject`
3. Add the `Authorization` header with the value `Bearer [YOUR_API_KEY]`
4. Include the required project information in the request body:

```json copy
{
  "name": "new test",
  "description": null,
  "abbreviation": null,
  "organizationId": null
}
```

**Request Body Parameters:**
- `name` (required): The name of your project
- `description` (optional): A description of your project
- `abbreviation` (optional): A short abbreviation for your project
- `organizationId` (optional): The ID of the organization this project belongs to

5. Send the request
